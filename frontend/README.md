# 🎨 Freelance Marketplace - Frontend

This is the frontend portion of the Freelance Marketplace project, built with [Next.js 15](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/). It provides a modern, responsive interface for clients and freelancers to collaborate.

---

## 🚀 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Environment Configuration
Create a `.env` file in this directory and specify your backend API URL:
```env
NEXT_PUBLIC_API_URL=http://localhost:4080
```

### 3. Run the Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

---

## 🛠️ Features

- **Modern UI Architecture:** Leverages Next.js 15 App Router for optimized performance and SEO.
- **Google OAuth Integration:** One-click login and signup flow directly integrated into the auth UI.
- **Dynamic Dashboards:** Specialized workspaces for Clients (Post Jobs, Manage Bids) and Freelancers (Browse Jobs, Track Proposals).
- **Professional Identity Hub:** Intuitive UI for uploading resumes, managing skills, and auditing professional portfolios.
- **Responsive Animations:** Powered by Framer Motion for a premium, interactive user experience.

---

## 📂 Folder Structure

- `src/app/`: Page-level components, global layouts, and route handlers.
- `src/components/`:
  - `UI/`: Stateless, reusable components (Buttons, Modals, Loaders).
  - `Layout/`: Global structural components (Navbar, Footer, Sidebar).
  - `Home/`: Sections and features specific to the landing page.
- `public/`: Static assets, brand logos, and media icons.
