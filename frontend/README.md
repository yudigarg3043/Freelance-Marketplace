# 🎨 Freelance Marketplace - Frontend

This is the frontend portion of the Freelance Marketplace project, built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/).

---

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

---

## 🛠️ Features

- **Modern UI:** Built with Tailwind CSS for a sleek and responsive design.
- **Client & Freelancer Flow:** Dedicated pages for posting jobs, submitting bids, and managing dashboards.
- **Form Validation:** Integrated with standard React practices.
- **API Integration:** Seamlessly communicates with the Node.js/Express backend.

---

## 📂 Folder Structure

- `src/app`: Page components and global layouts under the App Router.
- `src/components`: Reusable UI components.
- `public`: Static assets (images, icons, etc.).
